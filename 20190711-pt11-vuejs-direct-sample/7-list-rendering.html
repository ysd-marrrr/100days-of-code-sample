1. v-for で配列に要素をマッピングする

  • {{ index }} 番目: {{ item.text }}

2. オブジェクトの v-for

  • {{ index }} : {{ name }} - {{ value }}

3. 状態の維持(v-bind:key)

  • {{ item.text }}

4. オブジェクト変更検出の注意

  • {{ item }}

5. フィルタ/ソートされた結果の表示

  • 未ソート: {{ s }}

  • methodでソート: {{ s }}
